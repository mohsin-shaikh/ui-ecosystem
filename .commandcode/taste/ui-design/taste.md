# Ui Design
- Prefer compact, information-dense layouts over big boxes. Keep cards small and meaningful. Respect user's time - they should quickly understand personality and values. Use visual elements (images as overlays, backgrounds) and small interactive animations rather than large static boxes. Confidence: 0.85
- Prefer interactive, gamified experiences over static content displays. Transform informational sections into playable games. Confidence: 0.75
- Ensure smooth animations and transitions for interactive elements. Avoid jerky or choppy movement. Confidence: 0.70
- For portfolio/personal sites: Use sidebar for informational content (past experiences, fun stuff, quick facts) and main section for highlights (current work, what user cares about, personality). Confidence: 0.75
- Use expandable/interactive content blocks for sections with rich media like images. Make content explorable rather than static. Confidence: 0.75
- Incorporate visual storytelling with images and icons throughout the design. Use images to enhance personality and context. Confidence: 0.75
- For expandable content: Expand inline below the trigger element, not as a replacement. Keep the first/primary element visible (e.g., first image in a stack) and expand additional content underneath. Confidence: 0.75
