## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/hover-card.json
```

The component source is published as a registry payload at
`/r/hover-card.json`.

## Usage

```tsx
import { HoverCard, HoverCardTrigger, HoverCardContent } from "@/components/ui/hover-card";
```

```tsx
<HoverCard />
```

## Dependencies

`@base-ui/react`
