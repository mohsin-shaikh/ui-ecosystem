## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/scroll-area.json
```

The component source is published as a registry payload at
`/r/scroll-area.json`.

## Usage

```tsx
import { ScrollArea, ScrollBar } from "@/components/ui/scroll-area";
```

```tsx
<ScrollArea />
```

## Dependencies

`@base-ui/react`, `react`
