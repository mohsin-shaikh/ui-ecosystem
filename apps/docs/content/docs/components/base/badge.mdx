## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/badge.json
```

The component source is published as a registry payload at
`/r/badge.json`.

## Usage

```tsx
import { Badge, badgeVariants } from "@/components/ui/badge";
```

```tsx
<Badge />
```

## Dependencies

`@base-ui/react`, `class-variance-authority`
