## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/skeleton.json
```

The component source is published as a registry payload at
`/r/skeleton.json`.

## Usage

```tsx
import { Skeleton } from "@/components/ui/skeleton";
```

```tsx
<Skeleton />
```

## Dependencies

No external dependencies.
