## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/progress.json
```

The component source is published as a registry payload at
`/r/progress.json`.

## Usage

```tsx
import { Progress, ProgressTrack, ProgressIndicator, ProgressLabel, ProgressValue } from "@/components/ui/progress";
```

```tsx
<Progress />
```

## Dependencies

`@base-ui/react`
