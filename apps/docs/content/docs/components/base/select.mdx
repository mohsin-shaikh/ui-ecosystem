## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/select.json
```

The component source is published as a registry payload at
`/r/select.json`.

## Usage

```tsx
import { Select, SelectContent, SelectGroup, SelectItem, SelectLabel, SelectScrollDownButton, SelectScrollUpButton, SelectSeparator, SelectTrigger, SelectValue } from "@/components/ui/select";
```

```tsx
<Select />
```

## Dependencies

`@base-ui/react`, `lucide-react`, `react`
