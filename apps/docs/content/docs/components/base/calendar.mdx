## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/calendar.json
```

The component source is published as a registry payload at
`/r/calendar.json`.

## Usage

```tsx
import { Calendar, CalendarDayButton } from "@/components/ui/calendar";
```

```tsx
<Calendar />
```

## Dependencies

`lucide-react`, `react`, `react-day-picker`
