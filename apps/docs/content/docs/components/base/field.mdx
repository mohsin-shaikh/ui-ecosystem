## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/field.json
```

The component source is published as a registry payload at
`/r/field.json`.

## Usage

```tsx
import { Field, FieldLabel, FieldDescription, FieldError, FieldGroup, FieldLegend, FieldSeparator, FieldSet, FieldContent, FieldTitle } from "@/components/ui/field";
```

```tsx
<Field />
```

## Dependencies

`class-variance-authority`, `react`
