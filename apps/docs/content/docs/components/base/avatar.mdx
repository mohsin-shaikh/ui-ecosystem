## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/avatar.json
```

The component source is published as a registry payload at
`/r/avatar.json`.

## Usage

```tsx
import { Avatar, AvatarImage, AvatarFallback, AvatarGroup, AvatarGroupCount, AvatarBadge } from "@/components/ui/avatar";
```

```tsx
<Avatar />
```

## Dependencies

`@base-ui/react`, `react`
