## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/tabs.json
```

The component source is published as a registry payload at
`/r/tabs.json`.

## Usage

```tsx
import { Tabs, TabsList, TabsTrigger, TabsContent, tabsListVariants } from "@/components/ui/tabs";
```

```tsx
<Tabs />
```

## Dependencies

`@base-ui/react`, `class-variance-authority`
