## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/native-select.json
```

The component source is published as a registry payload at
`/r/native-select.json`.

## Usage

```tsx
import { NativeSelect, NativeSelectOptGroup, NativeSelectOption } from "@/components/ui/native-select";
```

```tsx
<NativeSelect />
```

## Dependencies

`lucide-react`, `react`
