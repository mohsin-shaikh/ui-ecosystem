## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/input.json
```

The component source is published as a registry payload at
`/r/input.json`.

## Usage

```tsx
import { Input } from "@/components/ui/input";
```

```tsx
<Input />
```

## Dependencies

`@base-ui/react`, `react`
