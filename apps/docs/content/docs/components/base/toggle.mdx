## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/toggle.json
```

The component source is published as a registry payload at
`/r/toggle.json`.

## Usage

```tsx
import { Toggle, toggleVariants } from "@/components/ui/toggle";
```

```tsx
<Toggle />
```

## Dependencies

`@base-ui/react`, `class-variance-authority`
