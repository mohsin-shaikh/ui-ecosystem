## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/radio-group.json
```

The component source is published as a registry payload at
`/r/radio-group.json`.

## Usage

```tsx
import { RadioGroup, RadioGroupItem } from "@/components/ui/radio-group";
```

```tsx
<RadioGroup />
```
