## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/direction.json
```

The component source is published as a registry payload at
`/r/direction.json`.

## Usage

```tsx
import { DirectionProvider, useDirection } from "@/components/ui/direction";
```

```tsx
<DirectionProvider />
```

## Dependencies

`@base-ui/react`
