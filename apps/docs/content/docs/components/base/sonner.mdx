## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/sonner.json
```

The component source is published as a registry payload at
`/r/sonner.json`.

## Usage

```tsx
import { Toaster } from "@/components/ui/sonner";
```

```tsx
<Toaster />
```

## Dependencies

`lucide-react`, `next-themes`, `sonner`
