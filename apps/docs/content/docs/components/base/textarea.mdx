## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/textarea.json
```

The component source is published as a registry payload at
`/r/textarea.json`.

## Usage

```tsx
import { Textarea } from "@/components/ui/textarea";
```

```tsx
<Textarea />
```
