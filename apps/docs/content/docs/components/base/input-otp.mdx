## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/input-otp.json
```

The component source is published as a registry payload at
`/r/input-otp.json`.

## Usage

```tsx
import { InputOTP, InputOTPGroup, InputOTPSlot, InputOTPSeparator } from "@/components/ui/input-otp";
```

```tsx
<InputOTP />
```

## Dependencies

`input-otp`, `lucide-react`, `react`
