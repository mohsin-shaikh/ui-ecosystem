## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/alert.json
```

The component source is published as a registry payload at
`/r/alert.json`.

## Usage

```tsx
import { Alert, AlertTitle, AlertDescription, AlertAction } from "@/components/ui/alert";
```

```tsx
<Alert />
```

## Dependencies

`class-variance-authority`, `react`
