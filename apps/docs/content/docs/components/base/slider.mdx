## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/slider.json
```

The component source is published as a registry payload at
`/r/slider.json`.

## Usage

```tsx
import { Slider } from "@/components/ui/slider";
```

```tsx
<Slider />
```

## Dependencies

`@base-ui/react`
