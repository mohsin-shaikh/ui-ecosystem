## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/button.json
```

The component source is published as a registry payload at
`/r/button.json`.

## Usage

```tsx
import { Button, buttonVariants } from "@/components/ui/button";
```

```tsx
<Button />
```

## Dependencies

`@base-ui/react`, `class-variance-authority`
