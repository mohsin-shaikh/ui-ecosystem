## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/command.json
```

The component source is published as a registry payload at
`/r/command.json`.

## Usage

```tsx
import { Command, CommandDialog, CommandInput, CommandList, CommandEmpty, CommandGroup, CommandItem, CommandShortcut, CommandSeparator } from "@/components/ui/command";
```

```tsx
<Command />
```

## Dependencies

`cmdk`, `lucide-react`, `react`
