## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/dialog.json
```

The component source is published as a registry payload at
`/r/dialog.json`.

## Usage

```tsx
import { Dialog, DialogClose, DialogContent, DialogDescription, DialogFooter, DialogHeader, DialogOverlay, DialogPortal, DialogTitle, DialogTrigger } from "@/components/ui/dialog";
```

```tsx
<Dialog />
```

## Dependencies

`@base-ui/react`, `lucide-react`, `react`
