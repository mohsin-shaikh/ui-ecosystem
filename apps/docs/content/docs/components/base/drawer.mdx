## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/drawer.json
```

The component source is published as a registry payload at
`/r/drawer.json`.

## Usage

```tsx
import { Drawer, DrawerPortal, DrawerOverlay, DrawerTrigger, DrawerClose, DrawerContent, DrawerHeader, DrawerFooter, DrawerTitle, DrawerDescription } from "@/components/ui/drawer";
```

```tsx
<Drawer />
```

## Dependencies

`react`, `vaul`
