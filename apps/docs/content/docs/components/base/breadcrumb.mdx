## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/breadcrumb.json
```

The component source is published as a registry payload at
`/r/breadcrumb.json`.

## Usage

```tsx
import { Breadcrumb, BreadcrumbList, BreadcrumbItem, BreadcrumbLink, BreadcrumbPage, BreadcrumbSeparator, BreadcrumbEllipsis } from "@/components/ui/breadcrumb";
```

```tsx
<Breadcrumb />
```

## Dependencies

`@base-ui/react`, `lucide-react`, `react`
