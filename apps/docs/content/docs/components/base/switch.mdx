## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/switch.json
```

The component source is published as a registry payload at
`/r/switch.json`.

## Usage

```tsx
import { Switch } from "@/components/ui/switch";
```

```tsx
<Switch />
```
