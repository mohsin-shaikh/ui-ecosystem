## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/spinner.json
```

The component source is published as a registry payload at
`/r/spinner.json`.

## Usage

```tsx
import { Spinner } from "@/components/ui/spinner";
```

```tsx
<Spinner />
```

`lucide-react`
