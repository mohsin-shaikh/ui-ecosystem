## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/label.json
```

The component source is published as a registry payload at
`/r/label.json`.

## Usage

```tsx
import { Label } from "@/components/ui/label";
```

```tsx
<Label />
```
