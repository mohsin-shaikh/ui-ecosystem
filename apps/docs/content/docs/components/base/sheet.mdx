## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/sheet.json
```

The component source is published as a registry payload at
`/r/sheet.json`.

## Usage

```tsx
import { Sheet, SheetTrigger, SheetClose, SheetContent, SheetHeader, SheetFooter, SheetTitle, SheetDescription } from "@/components/ui/sheet";
```

```tsx
<Sheet />
```

## Dependencies

`@base-ui/react`, `lucide-react`, `react`
