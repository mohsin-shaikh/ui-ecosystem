## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/button-group.json
```

The component source is published as a registry payload at
`/r/button-group.json`.

## Usage

```tsx
import { ButtonGroup, ButtonGroupSeparator, ButtonGroupText, buttonGroupVariants } from "@/components/ui/button-group";
```

```tsx
<ButtonGroup />
```

## Dependencies

`@base-ui/react`, `class-variance-authority`
