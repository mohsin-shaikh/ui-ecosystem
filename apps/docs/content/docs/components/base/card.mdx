## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/card.json
```

The component source is published as a registry payload at
`/r/card.json`.

## Usage

```tsx
import { Card, CardHeader, CardFooter, CardTitle, CardAction, CardDescription, CardContent } from "@/components/ui/card";
```

```tsx
<Card />
```

`react`
