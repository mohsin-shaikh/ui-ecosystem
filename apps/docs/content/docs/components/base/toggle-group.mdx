## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/toggle-group.json
```

The component source is published as a registry payload at
`/r/toggle-group.json`.

## Usage

```tsx
import { ToggleGroup, ToggleGroupItem } from "@/components/ui/toggle-group";
```

```tsx
<ToggleGroup />
```

## Dependencies

`@base-ui/react`, `class-variance-authority`, `react`
