## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/item.json
```

The component source is published as a registry payload at
`/r/item.json`.

## Usage

```tsx
import { Item, ItemMedia, ItemContent, ItemActions, ItemGroup, ItemSeparator, ItemTitle, ItemDescription, ItemHeader, ItemFooter } from "@/components/ui/item";
```

```tsx
<Item />
```

## Dependencies

`@base-ui/react`, `class-variance-authority`, `react`
