## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/checkbox.json
```

The component source is published as a registry payload at
`/r/checkbox.json`.

## Usage

```tsx
import { Checkbox } from "@/components/ui/checkbox";
```

```tsx
<Checkbox />
```

## Dependencies

`@base-ui/react`, `lucide-react`
