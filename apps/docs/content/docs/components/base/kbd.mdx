## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/kbd.json
```

The component source is published as a registry payload at
`/r/kbd.json`.

## Usage

```tsx
import { Kbd, KbdGroup } from "@/components/ui/kbd";
```

```tsx
<Kbd />
```

## Dependencies

No external dependencies.
