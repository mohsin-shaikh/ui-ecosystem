## Installation

```bash
npx shadcn@latest add https://ui-ecosystem.dev/r/carousel.json
```

The component source is published as a registry payload at
`/r/carousel.json`.

## Usage

```tsx
import { CarouselApi, Carousel, CarouselContent, CarouselItem, CarouselPrevious, CarouselNext, useCarousel } from "@/components/ui/carousel";
```

```tsx
<Carousel />
```

## Dependencies

`embla-carousel-react`, `lucide-react`, `react`
